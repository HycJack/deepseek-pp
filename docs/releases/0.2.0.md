## DeepSeek++ 0.2.0

0.2.0 将 DeepSeek++ 从“记忆 + Skill”扩展升级为完整的浏览器端工具平台，核心新增 MCP 工具系统和 Codex-style 自动化任务。

### 主要变化

- 新增 MCP 服务管理、工具发现、健康检查、调用历史、超时和结果大小限制。
- 支持 Streamable HTTP、HTTP POST、SSE、stdio bridge 和 Chrome Native Messaging MCP 传输。
- 手动聊天和自动化任务共享动态工具 schema、自动执行 MCP 调用，并把 `<tool_results>` continuation 回同一 DeepSeek 会话。
- 新增侧边栏自动化任务、独立会话、立即运行、cron/RRULE 调度、运行历史、PoW/auth 兼容和失败状态展示。
- 工具调用内核重构为 provider-neutral runtime，内置记忆工具和 MCP 工具复用同一套 descriptor、解析、过滤和历史恢复链路。
- 记忆系统新增 update/delete 工具，并优化相关记忆筛选、思考模式和工具执行折叠展示。
- 新增 `/skill` 自动补全面板、系统提示词预设、预设导入、Expert 模式切换、WebDAV 同步和 DeepSeek 页面自定义背景。
- 补充 README 截图、MCP 操作说明、mock 验证脚本、TypeScript 修复和正式版本发布 workflow。
