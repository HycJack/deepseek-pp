## DeepSeek++ 0.4.4

0.4.4 是 Shell MCP 商店安装体验修复版本，重点让通过浏览器商店安装的用户也能按侧边栏提示完成本机 Shell Host 配置。

### 主要变化

- 新增可发布的 `deepseek-pp-shell-host` npm installer，用户可通过 `npx deepseek-pp-shell-host install ...` 安装 Shell Native Host。
- Shell Host 现在安装到用户目录，不再依赖插件源码目录。
- installer 支持 Chrome、Edge、Chromium 和 Firefox，并提供 `install`、`status`、`uninstall` 命令。
- MCP 侧边栏的 Shell 提示改为面向商店用户的 `npx` 命令，并自动填入当前扩展 ID。
- 对 Native Host 已安装但扩展 ID 未授权的情况提供明确提示，避免把 `forbidden` 误判成未安装。
- README、Chrome Web Store 文案和 MCP 操作说明同步为用户安装路径，源码安装命令仅作为开发者入口保留。

### 验证

- `npm run compile`
- `npm run smoke:shell`
- `npm run smoke:mcp`
- `npm run verify:mcp:mock`
- `npm run verify:automation`
- `npm pack --dry-run --workspace packages/shell-host`
- `npm run build:all`
- `npm run zip:all`
