## DeepSeek++ 0.6.2

0.6.2 是对话导出和跨平台 Shell 体验增强版本，重点让用户可以本地归档 DeepSeek 对话，并提升 Windows 本机命令链路的稳定性。

### 主要变化

- 侧边栏新增「导出」页，可把当前账号可访问的 DeepSeek 对话导出为 JSON、Markdown 或可打印 HTML。
- 导出内容支持可读模式和原始模式，并记录附件引用、文件名、大小、状态和消息关联。
- Windows Shell Host 的路径解析和中文输出更可靠，减少本机命令执行时的乱码和找不到命令问题。
- 新增 Vitest 单元测试覆盖请求增强、MCP 传输、记忆工具、同步 schema、Shell policy 和导出流程。
- Release workflow 会在发布前校验版本一致性，并确认 Shell Host npm 包可见后再上传浏览器扩展资产。
