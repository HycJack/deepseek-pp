## DeepSeek++ 0.7.0

0.7.0 是项目上下文、交互式工具和侧边栏体验增强版本，让 DeepSeek++ 更像一个可持续工作的浏览器内 AI Agent 工作台。

### 主要变化

- 项目上下文可整理仓库、网页、本地文件和手工说明，并在需要时加入对话，减少重复粘贴背景。
- 可下载产物支持生成单文件或项目包，适合保存脚本、Markdown、JSON、HTML 或小型项目结构。
- 沙箱确认、Skill 草稿、记忆导入、保存片段和提示词控制继续收敛，让多步任务更容易检查和接管。
- 侧边栏对话、右键场景、官方 API Key 配置和工具结果恢复更完整，刷新后仍能保留更清晰的执行反馈。
- 新版本会在侧边栏提示更新内容，用户关闭后按版本记忆，不再重复打扰。
