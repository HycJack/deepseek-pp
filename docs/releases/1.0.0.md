## DeepSeek++ 1.0.0

1.0.0 是多模态媒体与侧边栏工作台成熟版本，让 DeepSeek++ 可以把图片、视频、MCP、本机工具和更清晰的管理界面放进同一条浏览器任务流。

### 主要变化

- **多模态媒体进入对话流**：安装多模态 Native Host 后，可在 DeepSeek 输入框添加图片或视频，先通过用户配置的 OpenAI / Gemini 分析媒体，再把分析结果带回当前消息继续生成。
- **多模态配置收口**：侧边栏新增多模态 API 设置，可配置 OpenAI / Gemini Key、模型和请求地址，并通过 MCP 页创建、测试和启用 `多模态` 预设。
- **媒体任务更稳**：多模态请求会按图片和视频数量扩展等待时间，减少较大媒体或多文件分析在返回前被过早中断的情况。
- **Artifact 展示更清楚**：生成的 artifact 结果会显示在折叠工具块外部，正文和可下载产物更容易区分和查看。
- **侧边栏管理更成熟**：设置页拆分为通用、API、外观、数据、提示词、语音和关于等子页，MCP、工具、Skill、自动化等页面补强加载状态和确认反馈。
- **界面一致性继续打磨**：工具卡片、Skill 弹窗、注入样式和侧边栏控件统一视觉密度、状态反馈和可访问性。
- **商店与隐私材料同步**：Chrome Web Store 文案和隐私政策补充多模态 API Key、用户选择媒体和本机 Native Host 的数据边界。

English:

- Adds a multimodal media workflow: after installing the Multimodal Native Host, users can attach images or videos in the DeepSeek input box and let DeepSeek++ analyze the media before continuing generation.
- Adds Multimodal API settings for OpenAI / Gemini keys, models, and request URLs, with MCP-page creation, testing, and enablement for the `Multimodal` preset.
- Multimodal requests now expand their wait time according to image and video count, reducing premature interruption for larger or multi-file analysis.
- Artifact results render outside the collapsed tool block so the answer and downloadable output are easier to distinguish and inspect.
- Settings are split into focused subpages, while MCP, Tools, Skills, and Automation pages get clearer loading states, confirmations, and visual consistency.
- Privacy and store materials now describe multimodal API keys, user-selected media, and local Native Host data handling.
